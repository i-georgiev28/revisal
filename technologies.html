Технологии

Frontend

HTML

  • Използвахме HTML, защото той беше основният език за структурирането на съдържанието в уеб страниците. Чрез HTML създадохме основни елементи като заглавия, параграфи, списъци, изображения, линкове и други. Той предостави логическа структура на документа, което беше необходимо за правилното му показване в браузъра и за по-добро разбиране от търсачките. Без HTML нямаше да можем да организираме съдържанието на страницата по начин, който да бъде полезен и достъпен за потребителите.

CSS

  • Използвахме CSS, за да стилизираме HTML елементите и да оформим външния вид на уеб страницата. Чрез CSS променихме цветовете, шрифтовете, размерите, разстоянията и други визуални характеристики на елементите. Това ни даде възможността да направим сайта по-привлекателен и лесен за използване. CSS също така ни позволи да управляваме разположението на елементите на страницата, така че тя да изглежда добре на различни устройства и екрани. С помощта на CSS добавихме анимации и ефекти, които направиха сайта по-динамичен и интересен за потребителите.
